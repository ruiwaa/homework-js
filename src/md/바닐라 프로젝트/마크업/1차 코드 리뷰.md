# 바닐라 프로젝트

## 영화 티켓 결제 페이지 마크업 및 스타일링

### 마크업

#### 1. 기본 구조

<div style = "padding: 20px">
기본 구조

```
<main>
 <section>영화 예매 정보</section>
 <section>할인 및 포인트</section>
 <section>최종 결제 수단</section>
</main>
<footer>최종 결제 금액</footer>
```

영화 결제 페이지는 메인 결제창과 최종 결제 금액인 푸터 이렇게 크게 구성을 잡고 마크업을 진행했다.

</div>

#### 2. 활성화 되었을 때 보이는 UI 마크업

#### 3. 접근성

1. h2 요소에 sr-only 추가
   각 섹션 요소의 제목이 스크린 리더에서만 읽혀지게,
2. aria-pressed, aria-label 추가
   할인 및 포인트, 최종 결제 수단 그리고 최종 결제 금액 영역에 있는 버튼에 접근성을 고려하여 aria-pressed 와 aria-label 을 추가했다. 그리고 최종 결제 금액 부분에 총 합계가 할인 금액에 따라서 금액이 달라지는 것을 알려주기 위해 aria-live = "polite" 를 추가해 변경 사항을 바로 알려주도록 했다.

3. 버튼 요소를 추가하여, 탭과 마우스로 접근이 가능하도록 마크업

### CSS 스타일링

각 영역의 레이아웃과 스타일링을 따로 구분하여, 먼저 각 영역의 레이아웃에 대한 스타일링의 코드를 작성하고, 후에 스타일을 적용하기 위한 코드를 작성하였다.
