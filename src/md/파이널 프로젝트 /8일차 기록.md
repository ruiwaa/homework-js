- 컴포넌트별 조건에 따라 input 입력 할 수 있게 상태 관리 만듦


### 🛠️ 사용 기술

- next.js

### 💡 배운 점

- next.js 환경인만큼 최대한 서버 컴포넌트를 활용하도록 하자. 서버 컴포넌트를 활용하려면 서버 액션도 능숙하게 사용할 수 있어야 된다는 것을 또 느꼈다. 

### 😵 어려웠던 점

- 각 input마다 타입이 다르고 또 해당 인풋에 맞는 조건의 입력값만 입력할 수 있도록 만들다 보니깐, 전부 상태 선언하고 클라이언트 컴포넌트로 만들어지는데, 일단 이렇게 만들고 나서 서버 컴포넌트로 만들 수 있도록 리펙토링을 해야 될 것 같다.
